我在练习Linux命令行

mkdir  创建文件夹

cd   进入文件夹

cat ----  显示文件内容

vi ---   进入编辑器编辑文件

pwd  查看当前目录

git version   查看git版本

git init  创建版本库

git add  --  把文件添加到版本库

git commit -m "fsdfsdf"  把文件提交到仓库  并附加说明

git status  查看版本库状态

git diff ---  查看修改内容

git log 查看版本历史节点

git log --prerry=oneline  简洁版历史版本节点

git reset --hard HEAD^  回退到上一版本节点
git reset --hard HEAD~100  回退到之前100个版本


git reflog   查看历史命令
